# AGXX CRO Calculator

A predictive CRO (Conversion Rate Optimization) and UX strategy platform built with React, TypeScript, and Vite.

## About AGXX Labs

AGXX Labs provides deterministic CRO and UX engineering using mathematical models to project ROI before commitment. We identify the highest-impact fixes and implement them directly.

**Website:** [agxxlabs.com](https://agxxlabs.com)

## Quick Start

### Prerequisites
- Node.js (v16+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/aliengrlxx-vegaprime/AGXX-CRO_Calculator.git
cd AGXX-CRO_Calculator

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will open at `http://localhost:5173`

## Project Structure

- `index.txs2` - Home page with hero, problem statement, methodology

- `vite.config.ts` - Vite build configuration
- `vercel.json` - Vercel deployment configuration

---

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Add New" → "Project"
4. Import your GitHub repository
5. Vercel will auto-detect settings and deploy

Your site will be live in minutes with automatic deployments on each push.
